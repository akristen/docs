Testcontainers Cloud subscriptions provide runtime minutes for cloud-based
integration test runs. Each Docker Core subscription includes a base allocation
of runtime minutes. You can pre-purchase additional minutes at discounted rates
or use on-demand minutes billed at the end of each monthly cycle. Minutes do
not roll over between billing periods.

To pre-purchase runtime minutes,
[contact Docker sales](https://www.docker.com/pricing/contact-sales/).

## Usage

Included base minutes reset each billing period and do not accumulate.
On-demand usage is billed at higher rates than pre-purchased capacity. If you
expect consistent usage beyond your subscription's included minutes,
pre-purchasing additional minutes reduces costs.

For an overview of Testcontainers Cloud, see the
[Testcontainers overview](/manuals/testcontainers.md).

## Upgrade runtime minutes

You can add Testcontainers Cloud runtime minutes in two ways:

- **Pre-purchase**: [Contact Docker sales](https://www.docker.com/pricing/contact-sales/)
  to pre-purchase runtime minutes at discounted rates.
- **On-demand**: Use unlimited runtime minutes on-demand with billing at the
  end of each monthly cycle. On-demand rates are higher than pre-purchased
  capacity.

## Deactivate

To cancel your Testcontainers Cloud add-on or adjust your pre-purchased
minutes, [contact Docker sales](https://www.docker.com/pricing/contact-sales/).

Your base minutes included in your Docker Core subscription remain available
after cancellation.
